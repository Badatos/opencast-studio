Opencast Studio

A web-based recording studio for Opencast.

This is free software under the terms of the MIT License developed by the ELAN e.V. in cooperation with the ETH Zürich. Please report bugs or submit new features on the project's GitHub page.

If you are interested in additional development or integrations into specific tools (e.g. LMS), please contact opencast-support@elan-ev.de.
